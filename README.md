# Task Manager

A simple Task Manager web application built using Spec-Driven Development with Kiro IDE.

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Persist tasks using localStorage
* Input validation for empty and invalid task titles
* Automatic restoration of tasks after page refresh

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Git
* GitHub
* Kiro IDE

## Development Process

This project was developed using a Spec-Driven Development workflow:

1. Requirements generation
2. Design creation
3. Task planning
4. Incremental implementation
5. Testing and validation

The implementation was divided into multiple phases:

* Project structure and static shell
* Validator module
* Task store
* localStorage persistence
* Renderer
* Event handlers
* Final testing and validation

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/JovaniBouKhalil/spec-driven-webapp.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No build tools or dependencies are required.
